# Tetris Java

Projeto desenvolvido em Java utilizando Swing para interface gráfica.

## Funcionalidades

* Tetris single-player
* Interface gráfica com Swing
* Sistema de colisão
* Rotação de peças
* Sistema de pontuação
* Níveis de dificuldade
* Velocidade progressiva
* Preview da próxima peça
* Ranking Top 10
* Salvamento em JSON
* Save/Load de jogo
* Restart do jogo

## Tecnologias

* Java
* Swing
* Gson
* JUnit 5

## Estrutura do Projeto

* `src/` → código-fonte
* `test/` → testes JUnit
* `docs/` → documentação Javadoc
* `data/` → arquivos JSON
* `release/` → versão executável

## Como Executar

### Pela IDE

Execute a classe:

```java
Main.java
```

### Pelo terminal

```bash
java -jar release/tetris.jar
```
